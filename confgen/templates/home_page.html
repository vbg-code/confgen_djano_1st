This script is used to deploy the pre-migration non-impacting configuration.
    - BGP will be configured with DENY-ALL in/out route-maps.
    For the migration window, a new impacting CR will be raised to:
    - admin shut the BGP peering to old WAN
    - configure the proper route-maps under BGP to new WAN

    Fill in the following fields to generate the configuration:


    {% csrf_token %} CR SNOW ID: (example: CRQ12345678)

    Project VRF: (example: ABC)

    VLAN ID: (hint: vlan range is 100 - 200)

    Subnet: (example: 10.11.22.0)

         Note: Mask is always/29 -- 255.255.255.248
